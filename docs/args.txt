1.Имя файла или директории для архивации:
    Указывается путь к файлу или папке, которую нужно заархивировать.
    -i --input "../path/to/file"
2.Имя выходного архива:
    Указывается имя и путь для создаваемого архива.
    -o --output "name_archive"
3.Тип архива:
    Некоторые архиваторы позволяют выбрать формат архива (например, ZIP, RAR, 7z, TAR ).
    -t --type "tar"
4.Уровень сжатия:
    Параметр, определяющий степень сжатия данных (например, минимальное, нормальное, максимальное).
    -l --compression-level 5
5.Метод сжатия:
    Некоторые архиваторы поддерживают разные алгоритмы сжатия (например, Deflate, LZMA, BZIP2).
    -m --compression-method deflate
6.Пароль для защиты архива:
    Если требуется зашифровать архив, пользователь может указать пароль.
    -p --password "1234"
7.Разделение архива на части (тома):
    Возможность разбить архив на несколько файлов заданного размера (например, для записи на CD/DVD или USB).
    -s --split-size 100M
8.Исключение файлов или директорий:
    Параметр для исключения определённых файлов или папок из архивации.
    -X --exclude "*.tmp"
9.Дополнительные опции:
    Например:
    1.сохранение структуры директорий
        1.1. -r --preserve-paths
    2.удаление исходных файлов после архивации
        2.1. -d --delete-after
    3.создание самораспаковывающегося архива (SFX) и т.д.
        3.1 -S --self-extracting
10.Режим работы:
    Некоторые архиваторы поддерживают разные режимы:
    обновление архива или тестирование целостности архива.
    1. Добавление файлов в существующий архив:
        1.1. -a (append) --append
        1.2. ./archiver -a existing_archive.zip
    2.Обновление архива:
        2.1. -u --update
        2.2.  ./archiver --update=archive.zip
    3.Тестирование целостности архива:
        3.1 -T (test) --test
